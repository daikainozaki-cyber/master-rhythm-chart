Master Rhythm Chart

コード進行を素早く入力・再生する学習支援ツール

入力方法

  • A-G ルート入力（インクリメンタル）
  • 1-16 メモリーリコール
  • / オンコード入力
  • Enter 確定・配置
  • ↑↓ 候補選択 / ライン移動
  • → テンション拡張（入力中）
  • Tab 補完
  • h = m7b5, q = quartal, maj = major

移動

  • ←→ 1拍移動
  • Tab 次の小節 / Shift+Tab 前の小節
  • Home 曲の先頭 / End 曲の末尾
  • ↑↓ ライン移動
  • コードをクリック → 編集モード

編集

  • Cmd+Z アンドゥ
  • Cmd+Shift+Z リドゥ
  • R リピート（直前のコード）
  • Cmd+D 複製
  • Del 削除（範囲選択時は一括）
  • Shift+←→ 範囲選択
  • Cmd+C 範囲コピー
  • Cmd+V 範囲ペースト（カーソル位置から）
  • Esc 範囲クリア / 再生停止

再生

  • Space 再生/停止

メモリー

  • 配置したコードは自動記録（最大16個）
  • メモリーボタンをクリック → リコール
  • Shift+Click → スロットクリア
  • ドラッグ → グリッドにドロップ配置
  • 長押しドラッグ → メモリー内並べ替え
